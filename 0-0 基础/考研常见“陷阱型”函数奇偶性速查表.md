# 奇函数（f(-x) = -f(x)）
- $\sin x$  
- $\tan x$  
- $\arcsin x$  
- $\arctan x$  
- $x^n$ （n 为奇数）  
- $\ln \dfrac{1-x}{1+x}$ （重点！常考）  
- $x e^{x^2}$ （因为 e^{x^2} 是偶函数）  

# 偶函数（f(-x) = f(x)）
- $\cos x$  
- $\sec x$  
- $\arccos x$  
- $x^n$ （n 为偶数）  
- $e^x + e^{-x} = 2\cosh x$  
- $\ln(1+x^2)$  
- $e^{x^2}$  
- $\dfrac{1}{1+x^2}$  

# 既非奇函数也非偶函数
- $e^x$  
- $\ln(1+x)$  
- $\sqrt{1+x}$  
- $x+1$  

---

## ⚠️ 常见考点小结
- 考研常用技巧：  
  - **奇函数在对称区间积分为 0**  
- $\int_{-a}^a f(x)\, dx = 0$

  - **偶函数在对称区间积分可化为两倍**  
- $\int_{-a}^a f(x)\, dx = 2\int_0^a f(x)\, dx$

- 典型套路：  
  - 遇到 $\ln \dfrac{1-x}{1+x}$ → 记得它是奇函数！  
  - 遇到 $\ln(1+x^2)$ → 记得它是偶函数！  
  - 遇到指数函数 $e^x$ → 既不是奇也不是偶，**别掉坑**。
